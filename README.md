### Pomodoro Timer

**Pomodoro Timer** is an easy to use, web-based Pomodoro Technique timer: https://timer.mhluska.com

Cross-browser testing is sponsored by [BrowserStack](https://www.browserstack.com).

#### Develop

```
bower install
gem install sass
npm install
script/build
script/serve # or `script/watch`
```

#### Deploy

```
script/deploy
```
